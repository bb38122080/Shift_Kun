```
cd Shift_Kun
```

## 使用方法
### 準備
このアプリケーションのGUIは、Pythonのワークスペース[Flet](https://flet.dev/)を利用している。  
GUI表示のためにターミナルで以下のコマンドを実行してください。
```
pip install flet
```

### メインメニュー
以下のコマンドをターミナルで実行するとアプリケーションを実行できます。
```
python main.py
```
  
実行するとメインメニューが起動します。  
このメインメニュー内のボタンをクリックすることで、各機能にアクセスできます。
